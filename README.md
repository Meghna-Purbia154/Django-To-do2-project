# Django To-Do Project

A simple Django web application to manage tasks and categories.

## Features
- Add, update, and delete tasks
- Organize tasks by category
- User authentication (signup/login/logout)
- Simple dashboard to view tasks
  
## Design
The application uses a **pastel color palette** for a soft and cute look.  
- Soft background colors for a calming experience  
- Buttons and highlights in complementary pastel shades  
- Minimalist layout for clarity and simplicity
